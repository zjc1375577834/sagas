本系统主要解决多外部系统交互的事务一致性问题，采用策略为sagas的理论方案，执行最终一致性。
使用方式见test Demo；
功能：支持同时调用多个外部接口，同步或者异步方式调用。
     对于不同事务，可以通过实现接口时候调整返回类型进行处理。如果为不可逆即不可回滚事务，则只能放在最后一个处理，不可同时处理两个，
     当系统异常宕机，需要执行sagasinitjob 再执行sagasrolljob保证数据正常，此时应该加一个调用开工（还未做）正常配置需要执行sagasIngjob和sagasRingJob，定时执行。
缺陷：目前job没有配置需要根据系统自带的job去实现。没有经过严格测试。
